# Extension Icons

请在此目录放置以下尺寸的插件图标：

- `icon16.png` - 16x16 像素
- `icon48.png` - 48x48 像素
- `icon128.png` - 128x128 像素

## 图标设计建议

- 使用简洁的设计，易于识别
- 使用 Parallel Resume 的品牌颜色（蓝色 #3b82f6）
- 确保在不同背景下都清晰可见
- PNG 格式，透明背景

## 在线图标生成工具

- [Canva](https://www.canva.com/) - 免费设计工具
- [Figma](https://www.figma.com/) - 专业设计工具
- [Icon Generator](https://icon.kitchen/) - 快速生成不同尺寸

## 临时解决方案

在开发阶段，你可以使用简单的彩色方块作为占位符图标。
